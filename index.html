O desenvolvimento web se trata da criação de um software, seja ele um sistema, um app, um portal ou um site, ques está conectado a uma rede, podendo ser internet ou intranet (uma rede interna). Software web são desenvolvidos para realizar operações em tempo real.

Hoje a maioria do que é desenvolvido é feito para web, permitindo conectividade, integração entre diferentes softwares e ações com resultados instantaneos.
